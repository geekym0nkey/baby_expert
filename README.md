
# BabyZen AI - 啟動說明書 👶✨

這是一個專為新手爸媽設計的 AI 助手。請按照以下步驟操作：

### 1. 安裝環境 (僅需一次)
請前往 [Node.js 官網](https://nodejs.org/) 下載並安裝 **LTS 版本**。

### 2. 設定 API 金鑰 (🔑 最重要的一步)
為了安全，我們不直接把密碼寫在程式裡，而是放在一個隱藏的「保險箱」檔案。

1. **申請金鑰**: 到 [Google AI Studio](https://aistudio.google.com/app/apikey) 點擊 **"Create API key"**，並複製代碼。
2. **找到模板**: 在專案資料夾中，找到檔案 `.env.example`。
3. **改名為保險箱**: 將 `.env.example` 重新命名為 **`.env`** (點開頭，沒有任何後綴)。
   - *注意*: 如果電腦不讓你改名（說必須有檔名），可以先打開記事本，貼上 `API_KEY=你的金鑰`，然後另存新檔，檔名輸入 `".env"` (包含引號)。
4. **貼上金鑰**: 確保檔案內長這樣：`API_KEY=AIzaSyBxxxx...`。

### 3. 如何啟動？
1. **開啟終端機**:
   - **Windows**: 在資料夾路徑輸入 `cmd` 按 Enter。
   - **Mac**: 右鍵點資料夾 ->「服務」->「從資料夾新建終端機」。
2. **輸入指令**:
   ```bash
   npm install  # 安裝 App 零件 (只需跑一次)
   npm run dev  # 啟動軟體
   ```
3. **開始使用**: 瀏覽器會自動跳出，或手動輸入 `http://localhost:3000`。

---

### 🌍 如何分享給家人朋友（像真正的 App 一樣）？
如果你希望家人在他們的手機點開就能用，不需要設定電腦，請參考以下建議：

1. **什麼是「部署」？**: 將這份程式碼上傳到雲端伺服器（推薦使用 **Vercel**，它是免費的）。
2. **如何設定金鑰？**: 
   - 當你把程式碼上傳到 Vercel 後，在它的網頁後台找到 **"Settings" -> "Environment Variables"**。
   - 在那裡新增一個名字叫 `API_KEY`，內容填入你的金鑰。
3. **手機加入主畫面**: 
   - 家人點開 Vercel 給你的網址後，在瀏覽器點選「**加入主畫面**」。
   - 它就會像一般 App 一樣出現在手機桌面，點開就能用，且**完全不需要**在手機上設定金鑰！

### ⚠️ 安全提醒
絕對不要把金鑰直接寫在 `geminiService.ts` 裡面再傳給別人。透過 `.env` 檔案或雲端平台的「環境變數」設定，才是最安全、專業的做法。
